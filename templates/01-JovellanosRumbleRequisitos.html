Requisitos del Sistema

Si tu PC no soporta Jovellanos Rumble, espabila y cómprate uno mejor.

Mínimo Recomendado Para Ejecutar Jovellanos Rumble

  • Sistema Operativo: Windows 10 64 bits
  • Procesador: Intel Core i5-6600K / AMD Ryzen 5 1600
  • Memoria: 16 GB de RAM
  • Gráficos: NVIDIA GeForce GTX 1060 3GB / AMD Radeon RX 580 4GB
  • DirectX: Versión 11
  • Almacenamiento: 50 GB de espacio disponible

Mínimo Fisiológico Recomendado Para Jugar Jovellanos Rumble

  • Tener 2 manos con al menos 3 dedos cada una
  • Tener 1 ojo
  • Saber leer y comprender (aunque solo sea para elegir la opcion del menu)
  • Tener hechos los deberes del colegio / instituto o tener un 4.9 de media en DAM
